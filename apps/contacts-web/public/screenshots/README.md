# PWA Screenshots

This folder should contain screenshots for the PWA install prompt and app stores.

## Required Images

### Desktop Screenshot
- **File**: `desktop-contacts.png`
- **Size**: 1280x720 pixels
- **Purpose**: Shows the app on desktop for install prompts

### Mobile Screenshot
- **File**: `mobile-contacts.png`
- **Size**: 750x1334 pixels (iPhone 8 size, works for most devices)
- **Purpose**: Shows the app on mobile for install prompts

## Guidelines

1. Capture actual app screenshots with realistic data
2. Use light mode for better visibility
3. Ensure no personal information is visible
4. Show the main contacts list with contact details
