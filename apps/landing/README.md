# Landing Page

Marketing and product showcase site for AppStandard Suite.

## Tech Stack

- React 19
- TanStack Router
- Tailwind CSS 4
- Vite 7

## Development

```bash
# From monorepo root
bun run dev:landing

# Or directly
bun run dev  # Starts on port 3010
```

## Structure

```
src/
├── components/     # UI components
├── routes/         # TanStack Router pages
└── main.tsx        # Entry point
```

## Features

- Product overview for Calendar, Tasks, and Contacts
- Responsive design
- Links to individual applications
